# Um número um voto

Um Número Um Voto (UNUV) é um aplicativo web, desenvolvido no padrão Nextjs, com o objetivo de identificar e
analisar favoritismo e concorrência política.

Através de um sistema automatizado, o app salva a preferência de diversos usuário e transforma em planilhas,
gráficos e estatísticas, entregando, por meio dessa iniciativa, um conjunto de informações no mínimo curiosas
e reflexivas.

Os usuários que se interessam em participar da pesquisa, conseguirão fazer isso contribuindo com um valor
simbólico para ajudar a manter os custos do site.
